Набор UI-автотестов для сайта https://selectel.ru
Проект выполнен в рамках тестового задания на стажировку QA.


Используемый стек:
- Python 3.12+
- pytest
- Playwright


Реализованные проверки:
1. Открытие главной страницы и проверка URL.
2. Проверка наличия header на странице.
3. Переход с главной страницы на страницу авторизации (my.selectel.ru/login) через ссылку «Войти».
4. Проверка поиска (если элемент присутствует — иначе тест пропускается).


Как запустить:
1. Создать виртуальное окружение "python3.12 -m venv venv", "source venv/bin/activate"
2. Установить зависимости "pip install -r requirements.txt"
3. Установить браузеры "python -m playwright install"
4. Запустить тесты "pytest" (Запуск с отображением браузера "pytest --headed")


Примечание:
Проект выполнен в рамках тестового задания. Тесты запускались и проверялись локально (macOS, Python 3.12).
